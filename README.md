# *so_long*

## Description
This is a graphical project that challenges students to create a simple 2D game using the MiniLibX library. The goal is to design a game with a player, collectibles, and obstacles while adhering to specific rules and constraints.

## Features
- **Graphical Display**: Utilizes the MiniLibX library for graphical representation.
- **Player Movement**: Allows the player to move within the game.
- **Collectibles and Obstacles**: Implements elements such as collectibles and obstacles to create a challenging environment.
- **Game Rules**: Follows specific rules, such as collecting all items before exiting the game.

## Getting Started
### Prerequisites
- The project is designed to be compatible with the 42 environment.
- MiniLibX library is required for graphical display.

### Installation
1. Clone the repository:

    ```bash
    git clone https://github.com/marzianegro/42-so_long.git
    ```

2. Navigate to the project directory:

    ```bash
    cd 42-so_long
    ```

3. Compile the project:

    ```bash
    make
    ```

### Usage
1. Run the game with a provided map:

    ```bash
    ./so_long map/defaultMap.ber
    ```

2. Interact with the game using arrow keys to move the player.

### Example
```bash
# Run the game with a provided map
./so_long map/defaultMap.ber
```
